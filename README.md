## 🚩 运行

1. 拉取代码
```bash
git clone https://github.com/Lukematch/admin-react.git
```
2. 安装依赖
```bash
// 前端
npm install
pnpm install
yarn install
// 后端
cd server
yarn install
```
3. 启动项目
```bash
// 前端
npm run dev
pnpm run dev
yarn start
// 后端
yarn start:dev
```
4. 访问地址
```bash
// 前端
http://localhost:6400
// 后端
http://localhost:3000
```
